> __Vapid__, a vacuous Redis implementation for connection tests.

###Install

```bash
$ npm install vapid [-g]
```

> __require__:

```javascript
var Vapid  = require( 'vapid' );
```

###Run Tests

```bash
$ cd vapid/
$ npm test
```
